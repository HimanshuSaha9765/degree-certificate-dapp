About CertChain

A decentralized degree certificate verification system built on Ethereum blockchain with AI fraud detection.

🔗
What is CertChain?

CertChain is a blockchain-powered academic certificate verification system that allows universities and colleges to issue tamper-proof digital certificates, and enables employers and the public to verify them instantly — without trusting any central authority.

Traditional paper certificates can be forged, altered, or backdated. CertChain eliminates this by storing certificates permanently on the Ethereum blockchain — where every record is public, immutable, and cryptographically secured.

How Blockchain Ensures Trust
📦
Immutable Storage
Once a certificate is written to the blockchain, it cannot be modified or deleted — not even by the issuing institution. Every record is permanent by design.
🌐
Decentralized Network
Data is stored across thousands of computers worldwide simultaneously. There is no single server that can be hacked, shut down, or manipulated.
🔑
Cryptographic Identity
Every issuer is identified by their unique MetaMask wallet address. Only the wallet that issued a certificate can revoke it — ensuring accountability without passwords.
🔍
Public Transparency
All certificate records are publicly readable by anyone — no login required. This creates complete transparency while keeping the system trustless and open.

AI Fraud Detection — How It Works

Our AI is not a neural network — it is a powerful rule-based intelligence engine that mimics how a trained fraud analyst thinks. It runs 7 checks on every certificate and assigns a Trust Score from 0–100.

1️⃣
Existence Check (−100 if fails)
First confirms the certificate exists on the blockchain. If not found, score goes to 0 immediately — it is completely invalid.
2️⃣
Revocation Check (−80 if revoked)
Checks if the issuing institution has revoked this certificate. Revoked certificates are permanently invalid regardless of other checks.
3️⃣
Expiry Check (−40 if expired)
If the issuer set an expiry date and that date has passed, the certificate is flagged as expired and score is reduced significantly.
4️⃣
Multi-Degree Pattern (−20 if suspicious)
Checks how many certificates exist for the same student name. If one person has 4 or more different degrees, the engine flags it as a suspicious pattern.
5️⃣
Bulk Issuing Detection (−15 if suspicious)
Checks how many certificates the issuer has issued in total. If an issuer has issued more than 50 certificates from one wallet, it raises a flag for further review.
6️⃣
Data Completeness Check (−10 if incomplete)
Verifies all required fields are present and non-empty. Missing student name, degree, or institution name is a red flag for data tampering or malformed records.
7️⃣
Year Validation (+5 bonus if recent)
Checks if the graduation year is realistic. Years before 1980 or in the future are flagged. Recently issued certificates get a small trust bonus.
📊 Trust Score Classification
✅ Safe (80–100) Certificate passes all checks. Highly trustworthy.
⚠️ Medium (50–79) Some patterns flagged. Verify with institution.
🚨 High Risk (0–49) Multiple issues detected. Do not trust without direct verification from institution.

Technology Stack
📝
Solidity
Smart contract language for Ethereum. Handles certificate storage, issuance, revocation and access control.
🦊
MetaMask + ethers.js
Wallet connection and blockchain interaction. ethers.js v5.7.2 handles all contract calls from the browser.
🌐
HTML + CSS + JavaScript
Pure vanilla frontend — no React or Node.js required. Works with any basic HTTP server or Live Server extension.
🔗
Sepolia Testnet
Ethereum test network used for deployment. Free test ETH available from faucets. No real money involved.
📄
jsPDF + html2canvas
PDF certificate generation in the browser. html2canvas captures the visual certificate, jsPDF exports it as a downloadable file.
📱
QRCode.js
Generates QR codes embedded in PDF certificates. Scanning the QR auto-opens the public verification page with ID pre-filled.
👥
User Roles & Access
🎓
Issuer — College / University
Sets up institution profile → Issues certificates on blockchain → Generates PDF with QR code and digital signature → Revokes if needed → Views full audit log of all actions
🔍
Verifier — Employer / Organization
Searches certificate by ID → Gets full details from blockchain → Receives complete AI fraud analysis report with Trust Score → Views detailed check results and risk flags
👤
Public User — Student / Anyone
No wallet needed → Enters Certificate ID or scans QR code → Gets simple Safe / Warning / Risky result → Can see basic certificate details → Cannot see advanced AI analysis
